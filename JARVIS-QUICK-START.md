# 🚀 JARVIS 2.0 Quick Start (TUI)

## Start

```bash
cd ~/Jarvis-2.0
./jarvis setup
jarvis
```

## Optional API

```bash
source venv/bin/activate
python3 -m src.api.main
```

- API: http://localhost:8000
- API Docs: http://localhost:8000/docs
